# Design Factory Group — Website

## Structure
- `client/` — React + Vite + Tailwind frontend
- `server/` — Node + Express + MongoDB backend

## Setup
1. `cd client && npm install`
2. `cd server && npm install`
3. Copy `.env.example` to `.env` in both `client/` and `server/`, fill in real values
4. Run backend: `cd server && npm run dev`
5. Run frontend: `cd client && npm run dev`

## Tech Stack
React, Vite, Tailwind CSS, Express, MongoDB (Atlas), Cloudinary, Resend, Leaflet, Framer Motion
